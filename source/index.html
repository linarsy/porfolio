Фронтенд разработчик

Алла Якушева

Летом 2017 г. я решила сменить специализацию и развиваться в области фронтенда. Для обучения я выбрала HTML Academy и успешно закончила 3 интенсива профессии «Фронтенд-разработчик». На данный момент прохожу стажировку в Лиге А (аутсорс по вёрстке сайтов от Academy). В моих планах на 2018 год много верстать, изучить React.js (или Vue.js) и пройти курс по JavaScript на Hexlet.

Основные навыки

  • HTML5 и CSS3

    Мои проекты отвечают современным стандартам: кроссбраузерность и валидность, семантика и доступность, прогрессивное улучшение и оптимизация. Я внимательно отношусь к деталям и использую PerfectPixel для максимального совпадения с макетом. Следование БЕМ и кодгайду делает мой код понятным и простым в поддержке.

  • JavaScript

    Уверенное знание основ JS: DOM-дерево, обработка событий, работа с сетью, структуры данных. Достаточно для реализации несложной интерактивности на веб-странице и прочный фундамент для изучения фреймворков.

  • Адаптивная верстка

    Адаптивные сетки на флексах, медиа-выражения, резина, адаптивная, ретиновая и векторная графика, кадрирование изображений, формат webp, векторные спрайты, оптимизация шрифтов и другое.

  • Инструменты

    Я работаю с системой контроля версий Git, веб-сервисами GitHub и GitLab для хостинга и публикации проектов, а также использую Travis CI для сборки и тестирования. Использование Node.js и npm позволяет мне автоматизировать стандартные задачи (Gulp) и увеличить скорость разработки (препроцессоры Sass, Less).

Мои работы

  • Мишка

    Главная / Каталог / Форма заказа

    Проект выполнен в рамках интенсива «HTML и CSS, уровень 2» в HTML Academy. «Мишка» прошел проверку по 53 критериям и получил высшую оценку у случайного наставника. В «Мишке» сложная адаптивная сетка (мобильная, планшетная и десктопная версии) и множество декоративных элементов. В работе над проектом я использовала БЕМ, Gulp, синтаксис SCSS, SVG-спрайты и ретинизацию изображений.

  • Кекстаграм

    kekstagram.ru

    «Кекстаграм» — сервис просмотра изображений, написан в рамках интенсива JavaScript в HTML Academy. Реализован следующий функционал: загрузка данных с удаленного сервера, фильтрация фотографий, загрузка нового изображения на сайт, изменение маштаба, применение эффектов и выбор глубины эффекта, валидация формы и отправка данных на сервер. Все критерии проверки выполнены мной на 100%.

  • Девайс

    Главная / Каталог

    Проект выполнен на интенсиве «HTML и CSS, уровень 1». По итогам проверки я набрала 100 баллов из 100. Вот некоторые из критериев Академии: сайт идентично отображается в последних версиях браузеров Chrome, Firefox, Safari, Edge и IE 11, все интерактивные элементы имеют текстовое описание и доступны с клавиатуры, выполнена стилизация нестандартных элементов форм, вёрстка проходит тест на переполнение контентом.
